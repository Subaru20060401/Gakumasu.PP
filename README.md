# 学マス プロデュース評価予測ツール

学園アイドルマスター（学マス）の定期公演『初』レジェンドで、**編成・レッスン・スケジュール・試験スコアから最終ステータスと評価値・到達ランクを予測**するWebツール。

## 特徴

- アイドル・サポートカード（実データ169枚）をプランで絞り込み・重複防止つきで選択
- サポカのアビリティを**凸0〜4の実値**で反映（得意率・パラメータボーナス・各種発動アビ）
- レッスン（週4/7/12/14/16・通常/SP）の固定上昇値ベースで予測
- スケジュール（おでかけ/活動支給/相談 等）とドリンク・Pアイテム生成の相互作用を加味
- メモリー4個・チャレンジPアイテム3枠に対応
- サポカ別の貢献詳細・評価値内訳を表示

## 技術構成

- Vite + TypeScript（バニラ、フレームワーク無し）
- 完全にクライアント側で動作（バックエンド不要の静的サイト）

## 開発

```bash
npm install
npm run dev     # http://localhost:5173
npm run build   # dist/ に本番ビルド
```

## データ出典

サポートカード・アイドル・Pアイテム等のデータは [学園アイドルマスターwiki（seesaa）](https://seesaawiki.jp/gakumasu/) を参照。カード画像は同wikiにリンクしています。

> 予測数値は暫定モデルによるものです。実際の最終ステータスは引き・イベント・プレイングに左右されます。
